# Portfolio Card 1

## Title
AI Job Search Automation Engine — Gmail to AI Scoring to CV Pack Workflow

## Short description
Designed an AI-assisted workflow that ingests job alerts from Gmail, filters and scores them using a structured rubric, and generates tailored CV packs using low-cost automation.

## Problem
Manual job tracking is slow, repetitive, and error-prone. Job alerts arrive across different sources, each with inconsistent formatting, and candidates waste time reviewing poor-fit roles or manually tailoring CVs.

## Solution
Built a local-first automation pipeline that connects Gmail job alerts to structured job scoring, Google Sheets tracking, and tailored CV generation. The system uses deterministic filters before AI calls to reduce cost and improve reliability.

## Tools / Stack
Gmail API, Google Sheets, Google Drive, n8n, Claude/LLM prompts, Python, Flask, python-docx, structured scoring rubric.

## Business value
- Reduced manual review effort by filtering poor-fit roles before AI processing
- Created a reusable scoring framework for decision-making
- Kept monthly operating cost low by using lightweight models and deterministic logic where possible
- Converted scattered job alerts into a structured operating system

## What this proves for clients
I can take a messy email/workflow problem, define the required fields, design the process, build a lightweight automation, and create a usable business tracker.

## Relevant Upwork services
- Gmail/email workflow automation
- Google Sheets automation
- AI-assisted data extraction
- Workflow scoring systems
- Lightweight operations dashboards

# Portfolio Card 3

## Title
RAG Evaluation & Safety Dashboard — AI PM Proof Project

## Short description
Created a PM-first RAG project that evaluates retrieval quality, citation validity, fallback behavior, latency, telemetry, and safety guardrails.

## Problem
Many AI chatbot/RAG demos appear functional but lack evaluation, observability, citation checks, and safety handling. This makes it difficult for teams to know whether the system is reliable enough for real users.

## Solution
Built a RAG microbuild ladder with multiple iterations: baseline RAG, caching, citations, citation validation, reranking, evaluation scorecards, telemetry dashboard, and safety guardrails.

## Tools / Stack
Python, Streamlit, vector retrieval concepts, RAG pipeline, evaluation scorecards, telemetry logs, citation validation, safety testing.

## Business value
- Makes AI chatbot quality measurable
- Helps compare retrieval changes before/after
- Surfaces latency, fallback, and reliability risks
- Adds PM-level thinking around trust, safety, and product readiness

## What this proves for clients
I understand AI product quality beyond demos: evaluation, trust, citations, safety, observability, and user-facing reliability.

## Relevant Upwork services
- RAG chatbot audit
